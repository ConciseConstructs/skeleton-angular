Deployment Platforms

Platforms are typically a collection of tools, services, and hardware infrastructure needed to serve and run modern websites and applications. There are several providers in this space that your website or mobile app can be deployed to. If you are unsure or have no preference for which provider you would like to run your application on, AWS (Amazon Web Services) is widely considered to be the leader and most robust.
